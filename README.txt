# 微信点餐互动程序 - 部署指南

## 概述
这是一个适合零基础用户的微信点餐互动程序，无需服务器，基于静态网页+本地存储实现。

## 功能特点
1. 用户界面：查看菜品、选择日期、点餐、查看点餐记录
2. 管理员界面：添加/编辑菜品、查看所有点餐记录、修改密码
3. 支持未来7天日期选择（可多选）
4. 支持B站/小红书视频链接
5. 数据自动保存到本地

## 文件说明
- index.html: 主页面文件
- style.css: 样式文件
- config.js: 配置文件
- app.js: 功能代码文件
- README.md: 本说明文件

## 快速开始
1. 下载所有文件到同一个文件夹
2. 用浏览器打开index.html
3. 初始管理员密码：admin123
